# MailChimp Comment

WordPress Plugin

By Philipp Rackevei

Version 1.0.0

License: see license.txt file

## Plugin Description

Adds a checkbox to a WordPress comment form so people can subscribe to your MailChimp list.

## Installation

* Edit the options in `mailchimp-comment.php` line 19-26
* Upload `mailchimp-comment.php` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
